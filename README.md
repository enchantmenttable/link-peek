Install via [Chrome Web Store](https://chromewebstore.google.com/detail/marginalia-link-peek-page/lemnmefmpdfpdepdhkmlelamdccdfbmm)

This extension lets you preview a link and add notes to a web page.

I built this because I kept losing my place while researching: opening links in many tabs, diving deep into a thread then forgetting where exactly I'm supposed to go, plus scattering notes across different apps.
